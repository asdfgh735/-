# Практическое задание

## Данные исполнителя:
- **ФИО:** Баймурзаев Амир Искандарбекович 
- **Группа:** 9КС-392
- **Стиль оформления кода:** Google C++ Style Guide

## Цель работы:
Отработать на практике принципы объектно-ориентированного проектирования, применение алгоритмов и контейнеров STL, механизмы файлового ввода-вывода, построение модульной архитектуры приложений, а также обеспечить корректную обработку пользовательского ввода в среде C++.

## Формулировка задачи:
Разработать консольную утилиту для автоматизации складского учёта. Приложение должно поддерживать полный цикл работы с данными: добавление, редактирование и удаление записей, поиск и фильтрацию по заданным критериям, сортировку списка, вычисление аналитических метрик, а также экспорт и импорт базы данных в текстовом формате (TXT).

## Технические требования:
- **Стандарт языка:** C++17 (или новее)
- **Компилятор:** Любой совместимый
- **Хранение данных в памяти:** `std::vector` или `std::list`
- **Формат внешнего файла:** Текстовый документ с фиксированным разделителем полей
- **Обработка некорректного ввода:** Исключить аварийное завершение программы (допускается принудительное завершение при ошибке ввода, но с вычетом 2 баллов)
- **Оформление исходного кода:** Строгое соответствие выбранному официальному гайдлайну
